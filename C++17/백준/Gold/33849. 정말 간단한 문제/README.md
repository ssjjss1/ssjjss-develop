### 문제 설명

<p>수열 $x_i$와 $y_i$가 주어진다. 아래 간단한 질문에 답을 해보자.</p>

<ul>
	<li>$x_i$, $y_i$의 $l$번째 원소부터 $r$번째 원소까지의 합의 비 $\frac{\sum_{i = l}^{r} y_i}{\sum_{i = l}^{r} x_i}$가 최대가 될 때의 값과 그때의 구간 길이의 최댓값을 구하라.</li>
</ul>

### 입력 

 <p>첫 번째 줄에 수열의 길이 $n$이 주어진다. ($1 \leq n \leq 500 \, 000$)</p>

<p>두 번째 줄에 수열 $x_i$의 원소 $n$개가 공백으로 구분되어 주어진다. ($1 \leq x_i \leq 10^9$)</p>

<p>세 번째 줄에 수열 $y_i$의 원소 $n$개가 공백으로 구분되어 주어진다. ($1 \leq y_i \leq 10^9$)</p>

<p>입력으로 들어오는 값은 모두 정수이다.</p>

### 출력 

 <p>첫 번째 줄에 비의 최댓값을 출력한다. 기약분수로 나타냈을 때 $\frac{A}{B}$라면 $A$와 $B$를 공백으로 구분하여 출력한다.</p>

<p>두 번째 줄에 비가 최대가 될 때의 구간 길이의 최댓값을 출력한다.</p>
